


## Howdy, I am bessy!
I'm an Information Technology student, and I gradute with a Bachelors degree in 2026!

My GitHub has many projcets, so here are the highlights:

🌐 [TEAW Website](https://github.com/besser435/TEAW-Website) - It's a website ([link](https://toendallwars.org/players)) for a Minecraft server. It has live updates from the server, and hosts a bunch of important information.

🤖 [TEAW API](https://github.com/besser435/TEAW-API) - It's a Java Plugin which collects data about a Minecraft server, and exposes that information as an HTTP API. It's what powers the TEAW Website!

🧸 [Player Plushies](https://github.com/besser435/Player-Plushies) - This is an upcoming NeoForge mod for Minecraft. You will be able to make cute little plushies, and customize them with player skins. It is very much still a WIP though!  

⏲️ [Lightbeam S3](https://github.com/besser435/Lightbeam-S3) - Have you ever wanted a clock with its display in binary? No? Ok, well, I made one anyways. It has a custom PCB that uses RGB LEDs for the bits,
an ambient light sensor to adjust the display brightness, and WiFi/GPS connectivity to set the time. It's written with CircuitPython and as it's one of my older projects, the code is pretty messy.
